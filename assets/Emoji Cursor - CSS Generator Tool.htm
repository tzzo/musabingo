Emoji Cursor

Use emojis as mouse cursors by combining SVG with CSS.

Steps:

  1. Choose your preferred emoji from the list below or provide your own.
  2. Adjust the size.
  3. Hover over the preview box to see the result.
  4. Copy the CSS and place it in your website stylesheet.

Options

40
Above 50 can cause blurriness and browser glitches.
 

Hover to Preview

Notes:

  • Chrome, Firefox, Edge 79+ & Safari only. No support for IE or pre-Chromium Edge.
  • Above size 50 introduces blurriness and sometimes glitches on Mac.
  • Greater than 32x32 may be unsupported in browsers.
  • For more information on changing your website cursors with CSS, please refer to this MDN article.
  • Preview doesn't work in safari after switching emoji, but copied CSS should still work.
  • Two or three text characters can be used instead of an emoji. Use the fill option in the SVG source to change the text colour.
  • Thanks to CoderPi for the inspiration.
  • Adding SMIL animation does appear to work in chrome while moving the cursor. Try hovering over the 'Emoji Cursor' heading at the top.
